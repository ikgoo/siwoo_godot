# 게임 실행 가이드

## 빠른 시작

1. **Godot 4.5 이상 설치**
   - [Godot 공식 사이트](https://godotengine.org/)에서 다운로드

2. **프로젝트 열기**
   - Godot 엔진 실행
   - "Import" 클릭
   - `project.godot` 파일 선택
   - "Import & Edit" 클릭

3. **게임 실행**
   - F5 키를 누르거나 우측 상단 "Play" 버튼 클릭
   - 처음 실행 시 메인 씬 선택 요청 시 `scenes/main.tscn` 선택

## 조작법

### 기본 이동
- **W**: 앞으로 이동
- **S**: 뒤로 이동
- **A**: 왼쪽으로 이동
- **D**: 오른쪽으로 이동
- **Space**: 점프
- **Shift**: 달리기
- **마우스**: 카메라 조준

### 무기 사용
- **좌클릭**: 발사
- **우클릭**: 조준 (ADS) - FOV 감소, 정확도 증가
- **R**: 재장전
- **1**: 주무기 (슬롯 1)
- **2**: 보조무기 (슬롯 2)
- **3**: 근접무기 (슬롯 3)
- **마우스 휠 위**: 다음 무기
- **마우스 휠 아래**: 이전 무기

### 기타
- **ESC**: 마우스 해제 (다시 ESC로 캡처)

## 게임 내 무기 목록

### 슬롯 1 (주무기) - 기본: 돌격소총
- **돌격소총**: 자동 연사, 30발, 중간 데미지
- **점사소총**: 3점사, 30발, 높은 정확도
- **AWP**: 저격총, 5발, 매우 높은 데미지
- **샷건**: 8발 동시 발사, 근거리 고데미지

### 슬롯 2 (보조무기) - 기본: 권총
- **권총**: 반자동, 12발, 빠른 재장전
- **UZI**: 자동 연사, 25발, 낮은 데미지

### 슬롯 3 (근접무기) - 기본: 주먹
- **주먹**: 기본 근접 공격
- **단검**: 중간 데미지, 빠른 공격
- **카타나**: 높은 데미지, 느린 공격

## 무기 변경하기 (개발자용)

게임 실행 중 무기를 변경하려면 `scripts/main.gd`의 `setup_weapons()` 함수를 수정하세요:

```gdscript
func setup_weapons():
    # 슬롯 0 (주무기) - 원하는 무기로 변경
    var weapon = weapon_scenes["assault_rifle"].instantiate()  # 다른 무기명으로 변경 가능
    weapon_manager.add_weapon(weapon, 0)
    
    # 사용 가능한 무기명:
    # "assault_rifle", "burst_rifle", "awp", "shotgun"
    # "pistol", "uzi"
    # "fist", "knife", "katana"
```

## UI 정보

### 화면 구성
```
                    [크로스헤어]
                        +


[HP: 100]                        [30 / 120]
[████████]                       [돌격소총]
```

- **왼쪽 아래**: HP 바 (색상이 HP에 따라 변화)
- **오른쪽 아래**: 탄약 정보 (현재탄창 / 예비탄약)
- **오른쪽 아래 하단**: 현재 무기 이름
- **화면 중앙**: 조준점 (크로스헤어)

### 피격 효과
- 데미지를 받으면 화면 테두리가 빨갛게 번쩍임
- 허수아비를 공격하면 데미지 숫자가 3D로 표시됨

## 테스트 맵 정보

기본 테스트 맵에는:
- 50x50 크기의 바닥
- 3개의 벽
- 3개의 허수아비 타겟 (무한 HP)

허수아비를 쏘면 데미지 숫자가 위로 떠오르며 표시됩니다.

## 문제 해결

### 마우스가 움직이지 않음
- ESC를 눌러 마우스를 캡처하세요
- 게임 실행 후 자동으로 마우스가 캡처됩니다

### 무기가 발사되지 않음
- 탄약을 확인하세요 (R키로 재장전)
- 발사 모드를 확인하세요 (일부 무기는 반자동)

### 낙하 데미지
- 높은 곳에서 점프하면 데미지를 받습니다
- 낙하 속도가 -15 이상일 때 데미지 발생
