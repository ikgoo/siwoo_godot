# FPS 멀티플레이 게임

Godot 4.5 엔진으로 제작된 1인칭 슈팅 게임입니다. 멀티플레이 확장을 고려하여 설계되었습니다.

## 주요 기능

### 플레이어 시스템
- **이동**: WASD 키로 이동
- **점프**: Space 키
- **달리기**: Shift 키
- **마우스 조준**: 1인칭 시점 (Yaw/Pitch 회전)
- **HP 시스템**: 100 HP, 피격 시 화면 효과
- **낙하 데미지**: 높은 곳에서 떨어질 시 데미지

### 무기 시스템

#### 슬롯 1 (주무기)
- **돌격소총**: 30발, 자동발사, 중간 데미지
- **점사소총**: 30발, 3점사, 높은 정확도
- **AWP**: 5발, 단발, 높은 데미지
- **샷건**: 8발, 8발 동시 발사

#### 슬롯 2 (보조무기)
- **권총**: 12발, 반자동
- **UZI**: 25발, 자동발사, 빠른 연사

#### 슬롯 3 (근접무기)
- **주먹**: 기본 근접 공격
- **단검**: 중간 데미지
- **카타나**: 높은 데미지, 긴 범위

### 무기 교체
- **마우스 휠**: 무기 순환 (위/아래)
- **숫자키 1,2,3**: 각 슬롯 직접 선택

### 조작법
- **W/A/S/D**: 이동
- **Space**: 점프
- **Shift**: 달리기
- **마우스 이동**: 조준
- **좌클릭**: 발사
- **우클릭**: 조준 (ADS)
- **R**: 재장전
- **1/2/3**: 무기 슬롯 선택
- **마우스 휠**: 무기 교체
- **ESC**: 마우스 해제

### UI 시스템
- **HP 바**: 왼쪽 아래, 색상 변화 (초록→노랑→빨강)
- **탄약 표시**: 오른쪽 아래, "현재탄창 / 예비탄약" 형식
- **무기 이름**: 현재 장착 무기 표시
- **크로스헤어**: 화면 중앙
- **데미지 인디케이터**: 피격 시 붉은 화면 효과

### FPS 핵심 기능
- **조준 정조준 (ADS)**: FOV 감소, 정확도 증가
- **이동 패널티**: 이동/점프 중 탄퍼짐 증가
- **반동 시스템**: 발사 시 카메라 흔들림
- **Head Bob**: 걷기/달리기 시 카메라 흔들림
- **레이캐스트 히트스캔**: 즉시 명중 판정

### 타겟 시스템
- **허수아비**: 무한 HP 타겟
- **데미지 숫자**: 3D 공간에서 위로 떠오르며 표시
- **피격 효과**: 데미지 파티클 (구현 예정)

## 프로젝트 구조

```
fps-multi/
├── scenes/
│   ├── player/          # 플레이어 씬
│   ├── weapons/         # 무기 씬들 (10개)
│   ├── ui/              # HUD, 데미지 숫자
│   ├── targets/         # 허수아비 타겟
│   └── main.tscn        # 메인 게임 씬
├── scripts/
│   ├── player/          # 플레이어 컨트롤러, 카메라
│   ├── weapons/         # 무기 베이스, 매니저, 데이터
│   ├── ui/              # HUD, 데미지 숫자
│   └── targets/         # 타겟 스크립트
└── resources/
    └── weapons/         # 무기 데이터 리소스 (10개)
```

## 멀티플레이 고려사항

이 프로젝트는 향후 멀티플레이 확장을 위해 다음과 같이 설계되었습니다:

- **독립적인 플레이어 씬**: 인스턴싱 가능
- **명확한 상태 관리**: 무기 상태, HP, 위치를 변수로 관리
- **서버 검증 준비**: 데미지 계산이 별도 함수로 분리
- **GameManager 싱글톤**: 게임 상태 중앙 관리
- **Signal 기반 통신**: 느슨한 결합으로 네트워크 동기화 준비

## 기술 스택

- **Godot 4.5**
- **GDScript**
- **CharacterBody3D**: 플레이어 물리
- **RayCast3D**: 총알 히트 판정
- **GPUParticles3D**: 시각 효과 (구현 중)
- **CanvasLayer**: 2D UI
- **Label3D**: 3D 데미지 숫자

## 물리 레이어

- **레이어 1 (World)**: 바닥, 벽 등 환경
- **레이어 2 (Targets)**: 타겟, 적
- **레이어 3 (Player)**: 플레이어
