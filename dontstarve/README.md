# 타일셋 분할 도구

`tiles.png` 이미지를 13x12 그리드로 분할하여 개별 타일 이미지로 저장하는 파이썬 스크립트입니다.

## 설치 방법

1. Python 3.6 이상이 설치되어 있어야 합니다.
2. 필요한 라이브러리를 설치합니다:

```bash
pip install -r requirements.txt
```

## 사용 방법

```bash
python tile_splitter.py
```

## 결과

- `split_tiles/` 폴더에 1.png, 2.png, 3.png ... 형태로 156개의 타일 이미지가 생성됩니다.
- 타일은 왼쪽 위부터 오른쪽 아래로 순서대로 번호가 매겨집니다.

## 입력/출력 예시

- 입력: `dontstarve/resorce/tile/tiles.png` (13x12 타일셋)
- 출력: `split_tiles/1.png`, `split_tiles/2.png`, ..., `split_tiles/156.png`

## 주요 기능

- **자동 타일 크기 계산**: 원본 이미지 크기를 기준으로 각 타일의 크기를 자동 계산
- **순차적 번호 부여**: 1부터 시작하는 연속된 번호로 파일명 생성
- **오류 처리**: 파일 없음, 권한 오류 등 다양한 예외 상황 처리
- **진행 상황 표시**: 각 타일 저장 과정을 실시간으로 출력
